# searchbookservice

## Project setup
```
node install (https://nodejs.org/ko/download/) - v10.16.0 개발 진행
npm install --global yarn
yarn install
```

### Run
```
yarn run serve 

Site > http://localhost:8081/
```

### OpenSource
```
vue : FE 프레임워크 개발 용도
element-ui : UI 개발 용도
axios : HTTP 통신 개발 용도
```
