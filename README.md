# searchbookservice

## Project setup
```
node download > <a href="https://nodejs.org/ko/download/">https://nodejs.org/ko/download/</a>
yarn install
```

### Run
```
yarn run serve 

Site > http://localhost:8081/
```

### OpenSource
```
vue : FE 프레임워크 개발 용도
element-ui : UI 개발 용도
axios : HTTP 통신 개발 용도
```
